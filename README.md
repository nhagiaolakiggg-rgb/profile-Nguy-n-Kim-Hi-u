# Portfolio - Trần Minh Tuấn

Website portfolio giới thiệu về hành trình của Trần Minh Tuấn - Nam sinh Chuyên Tin đạt Top 7 Đại học Hoa Kỳ (UChicago).

## Giới thiệu

Website này kể về câu chuyện "Ba Đôi Mắt" - một bài luận gây ấn tượng với hội đồng tuyển sinh Chicago, và hành trình sử dụng công nghệ để tạo ra tác động tích cực cho xã hội.

## Tính năng

- Design hiện đại với hiệu ứng glass morphism
- Responsive design, tương thích mọi thiết bị
- Smooth scroll animations
- Timeline về quy trình làm việc
- Giới thiệu các dự án Tech-for-Good

## Công nghệ sử dụng

- HTML5
- Tailwind CSS (CDN)
- Font Awesome Icons
- Vanilla JavaScript

## Cách sử dụng

Chỉ cần mở file `index.html` trong trình duyệt web.
